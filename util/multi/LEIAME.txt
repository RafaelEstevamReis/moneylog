MoneyLog - http://aurelio.net/moneylog
/util/multi
---------------------------------------------------------------------

Arquivos de teste para a função de usar vários arquivos TXT, de
maneira isolada ou simultânea.

Coloque as seguintes linhas no config.js:

dataFiles = [
	'util/multi/1.txt',
	'util/multi/2.txt',
	'util/multi/3.txt',
	'util/multi/4.txt',
	'util/multi/5.txt',
	'*'
];

Cada arquivo pode ser carregado separadamente. Se escolher o
asterisco, todos são carregados simultaneamente, como se fossem
apenas um único arquivo.

O asterisco pode ser colocado como primeiro item (default),
fazendo com que todos os TXT sejam carregados já no início.

Observar o comportamento das tags (selecionadas ou não), das
outras opções de filtragem e do tipo do extrato.

Usar o botão de Reload para verificar seu funcionamento.

Se usar uma vírgula extra no último item, o IE tenta carregar
o item undefined.
