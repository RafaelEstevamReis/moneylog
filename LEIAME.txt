=============================================================================
                    MoneyLog Experience — Versão BETA
=============================================================================

Para saber detalhes sobre instalação, configuração e novidades desta versão
beta, acesse: http://aurelio.net/moneylog/beta.html

Para acompanhar as novidades de perto, siga-me no twitter: @oreio


-----------------------------------------------------------------------------
DOWNLOAD
-----------------------------------------------------------------------------

  cd $HOME
  svn checkout http://moneylog-dev.googlecode.com/svn/trunk/ moneylog-dev

  Execute este comando somente uma vez.

-----------------------------------------------------------------------------
MIGRAÇÃO de dados do MoneyLog 3 para o Beta
-----------------------------------------------------------------------------

  Copie e cole seus lançamentos dentro do arquivo moneylog.txt


-----------------------------------------------------------------------------
ATUALIZAÇÃO
-----------------------------------------------------------------------------

  cd $HOME/moneylog-dev
  svn update

  Execute este comando sempre que quiser.


-----------------------------------------------------------------------------
DICAS
-----------------------------------------------------------------------------

* Crie um arquivo config.js para guardar suas configurações. O formato é o
  mesmo do início do moneylog.js.

* Use a opção dataFiles para trocar o nome do arquivo de dados:

     dataFiles = ['dados.txt'];

  ou para utilizar mais de um arquivo:

     dataFiles = ['pessoal.txt', 'profissional.txt'];

* Se tiver problemas com os acentos, converta o arquivo TXT para ISO-8859-1:

     iconv -f utf-8 -t iso-8859-1 moneylog.txt > moneylog-iso.txt

* Deixe todos os arquivos dentro da mesma pasta para evitar problemas.

* Abra o moneylog.html em seu navegador preferido e aproveite!
